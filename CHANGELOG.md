## v0.4.2 (2026-06-25)

### ✨ 新功能
- **初始化向导**：首次无设备时自动进入初始化向导，设备管理页可再次打开
- **设备详情页**：新增路由器详情视图，展示基础信息、系统状态、接口、存储和实时指标
- **设备识别增强**：连接设备页支持 OUI 厂商识别、主机名规则、用户自定义识别规则和设备类型建议
- **设备别名与图标**：支持固定名称、重要设备标记、品牌图标、设备类型图标和自定义图标上传
- **网络流量排行**：网络分析页新增终端流量排行，优先读取 nlbwmon / conntrack 等插件数据
- **访问保护**：新增管理员登录、JWT Cookie 会话、API/WebSocket 访问保护和 `OWM_AUTH_DISABLED` 开发开关
- **备份恢复**：系统设置页新增配置导出、配置导入和 SQLite 数据库备份下载
- **历史数据维护**：支持 3 / 7 / 14 / 30 天历史数据保留策略和立即清理过期历史数据

### 🐛 修复
- **初始化入口**：初始化向导不再固定显示在主菜单，仅在首次无设备或设备管理页中进入
- **仪表盘布局**：修复吞吐曲线模块和右侧信息模块不齐平的问题
- **设备密码**：编辑设备时密码留空不再覆盖原密码，设备列表 API 不返回密码字段

### 🎨 优化
- **响应式布局**：优化桌面、4K、Pad 和手机端的页面自适应表现
- **前端性能**：路由、ECharts 和图标资源拆分加载，降低首屏主包体积
- **图标可读性**：为设备图标增加浅色底板，提升黑色、灰色图标的可读性
- **SVG 资源**：压缩并规范品牌/设备 SVG 图标，支持分类搜索和后续扩展

### 🔧 其他
- **部署文档**：补充登录保护、`OWM_SECRET_KEY`、备份恢复、Docker 和 Unraid 环境变量说明

## v0.4.0 (2026-06-09)

### ✨ 新功能
- **网络地址卡片**：仪表盘新增“网络地址”区域，同时展示 WAN 地址和 LAN 地址
- **GitHub 自动打包**：新增根目录多阶段 `Dockerfile`，GitHub Actions 可直接构建并推送一体化 Docker 镜像到 GHCR

### 🐛 修复
- **WAN 地址来源**：外网/WAN IP 改为从 OpenWrt 接口状态读取，不再请求公网 IP 服务
- **系统日志**：修复日志接口因 SSH 返回对象字段不兼容导致 500 的问题
- **连接跟踪进度条**：低占用时显示真实百分比，并增加非零最小可见宽度

### 🎨 优化
- **仪表盘布局**：顶部指标调整为 `5 + 2` 布局，提升宽屏可读性
- **字体统一**：统一全局 UI 字体、数字字体和等宽字体使用规则
- **代码结构**：采集器改为注册表模式，后续新增采集模块更容易维护
- **Docker 构建上下文**：新增 `.dockerignore`，避免把本地数据库、虚拟环境和构建产物打进镜像上下文
