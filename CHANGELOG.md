# 📋 更新日志

## v0.4.1 (2026-06-18)

### 🔧 其他
- **本地发布流程**：新增 `scripts/release.sh`，自动同步版本号、生成发布提交、创建标签并推送到 GitHub
- **GitHub Release**：新增标签触发的 Release workflow，推送 `v*` 标签后自动生成 GitHub Release

## v0.4.0 (2026-06-09)

### ✨ 新功能
- **网络地址卡片**：仪表盘新增“网络地址”区域，同时展示 WAN 地址和 LAN 地址
- **GitHub 自动打包**：新增根目录多阶段 `Dockerfile`，GitHub Actions 可直接构建并推送一体化 Docker 镜像到 GHCR

### 🐛 修复
- **WAN 地址来源**：外网/WAN IP 改为从 OpenWrt 接口状态读取，不再请求公网 IP 服务
- **系统日志**：修复日志接口因 SSH 返回对象字段不兼容导致 500 的问题
- **连接跟踪进度条**：低占用时显示真实百分比，并增加非零最小可见宽度

### 🎨 优化
- **仪表盘布局**：顶部指标调整为 `5 + 2` 布局，提升宽屏可读性
- **字体统一**：统一全局 UI 字体、数字字体和等宽字体使用规则
- **代码结构**：采集器改为注册表模式，后续新增采集模块更容易维护
- **Docker 构建上下文**：新增 `.dockerignore`，避免把本地数据库、虚拟环境和构建产物打进镜像上下文

## v0.2.0 (2026-06-04)

### ✨ 新功能
- **系统分析页**：CPU 每核曲线、内存面积图、进程 TOP 10
- **连接设备页**：DHCP 租约列表、ARP 在线探测、设备备注
- **告警系统**：8 条可配置规则（CPU/内存/连接数/磁盘/负载/离线）
- **告警规则配置页**：阈值编辑、启用/禁用、重置默认
- **历史存储**：SQLite 时序表，支持 1h/6h/24h/7d 回看
- **流量历史曲线**：仪表盘支持时间范围选择，查看历史流量
- **三主题切换**：极夜🌙 / 暖阳☀️ / OLED🖤

### 🔧 其他
- GitHub Actions 自动构建 Docker 镜像到 ghcr.io
- GitHub 版本检查 + 更新提示
- Unraid Docker 模板（unraid-template.xml）
- Git tag 触发版号 Docker 构建

## v0.1.0 (2026-06-04)

### ✨ 初始版本
- **仪表盘**：CPU / 内存 / 温度 / 磁盘 / 负载 / conntrack 实时监控
- **实时流量曲线**：WAN 口上下行速率折线图
- **单位切换**：b/s ↔ B/s
- **采集间隔**：1s / 3s / 5s / 10s 可选
- **设备管理**：添加/删除 OpenWrt 设备（SSH 密码/密钥）
- **网络分析**：全接口列表、上下行总量、连接协议饼图、TCP 状态
- **WebSocket 实时推送**，毫秒级数据更新
- **暗色主题**，专业监控面板风格
- **Docker 一键部署**
- **全程由 AI 生成** 🎉

### 📝 完整功能
- 采集：CPU / 内存 / 负载 / 温度 / 磁盘 / 网络接口 / conntrack / DHCP 租约 / ARP 表
- 前端：Vue 3 + ECharts + TailwindCSS
- 后端：FastAPI + asyncssh + SQLAlchemy + SQLite
