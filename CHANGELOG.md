## v0.3.0 (2026-06-05)

### ✨ 新功能

## v0.2.0 (2026-06-04)

### ✨ 新功能
- **系统分析页**：CPU 每核曲线、内存面积图、进程 TOP 10
- **连接设备页**：DHCP 租约列表、ARP 在线探测、设备备注
- **告警系统**：8 条可配置规则（CPU/内存/连接数/磁盘/负载/离线）
- **告警规则配置页**：阈值编辑、启用/禁用、重置默认
- **历史存储**：SQLite 时序表，支持 1h/6h/24h/7d 回看
- **流量历史曲线**：仪表盘支持时间范围选择，查看历史流量
- **三主题切换**：极夜🌙 / 暖阳☀️ / OLED🖤

### 🔧 其他
- GitHub Actions 自动构建 Docker 镜像到 ghcr.io
- GitHub 版本检查 + 更新提示
- Unraid Docker 模板（unraid-template.xml）
- Git tag 触发版号 Docker 构建

## v0.1.0 (2026-06-04)

### ✨ 初始版本
- **仪表盘**：CPU / 内存 / 温度 / 磁盘 / 负载 / conntrack 实时监控
- **实时流量曲线**：WAN 口上下行速率折线图
- **单位切换**：b/s ↔ B/s
- **采集间隔**：1s / 3s / 5s / 10s 可选
- **设备管理**：添加/删除 OpenWrt 设备（SSH 密码/密钥）
- **网络分析**：全接口列表、上下行总量、连接协议饼图、TCP 状态
- **WebSocket 实时推送**，毫秒级数据更新
- **暗色主题**，专业监控面板风格
- **Docker 一键部署**
- **全程由 AI 生成** 🎉

### 📝 完整功能
- 采集：CPU / 内存 / 负载 / 温度 / 磁盘 / 网络接口 / conntrack / DHCP 租约 / ARP 表
- 前端：Vue 3 + ECharts + TailwindCSS
- 后端：FastAPI + asyncssh + SQLAlchemy + SQLite
